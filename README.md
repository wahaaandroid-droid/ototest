# MIDI Chord Rider

ブラウザ上でMIDIファイルを読み込み、曲中の一部パートだけをプレイヤーが演奏する感覚で遊べるReact/Vite製の音楽ゲームMVPです。

## セットアップ

```bash
npm install
npm run dev
```

本番ビルド:

```bash
npm run build
```

## 遊び方

1. `.mid` または `.midi` ファイルをアップロードするか、起動画面のサンプル曲を選びます。
2. 解析されたトラック一覧から、1つを `プレイヤー` にします。
3. 他のトラックは `自動演奏` または `ミュート` にできます。
4. `単音` または `コード` モードと、3段階の難易度を選び、ゲームを開始します。
5. 下部の大きなボタンをタイミングよくタップします。PCでは左から `A` `S` `D` `F` `J` `K` キーでも入力できます。

## WebAudioFontについて

音源はWebAudioFontのサンプルベース音源を利用します。初回再生時にWebAudioFont Playerと近い楽器プリセットを読み込みます。MIDI楽器名やプログラム番号に応じて、ピアノ、ギター、ベース、ストリングス、ドラムなどへ近似マッピングします。

## 注意点

- WebAudioFontのプリセットは外部URLから読み込むため、オフライン環境では音源ロードに失敗する場合があります。
- BPM変化や複雑なMIDI構造はMVPでは簡易対応です。
- 長押しノーツは押し始め判定を中心に扱い、離した時点でフェードアウトします。
- `public/midi/` 配下のサンプルMIDIはVercelでも配信されます。
